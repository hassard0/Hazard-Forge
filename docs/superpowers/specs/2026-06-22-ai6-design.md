## Design call: the VD6/NAV6/IK6 capstone mold — render-only float delegates over the frozen integer AI world
The deterministic-sim flagships all end this way (vd6_game, nav_render, ik6_render): the bit-exact integer world → a
one-line `…ToRenderInstances` delegate → the existing lit pipeline → a Metal-baked float golden. AI6 IS that for the AI
world:
- **The bridges are delegates, not new math.** `AgentToRenderInstances` delegates to the FROZEN
  `verdict::VerdictToRenderInstances` / `AppendMarkerCube` (`verdict.h:1027/987`) for the agent + player markers; the
  corridors use `nav::PathToWorldPolyline` (`navmesh.h:1262`) → the existing debug-line pipeline; the navmesh overlay
  uses `nav::PolyMeshToRenderMesh` (`navmesh.h:1230`) → the existing transparent pipeline. The agent's CHASE/PATROL
  state tints its marker. The integer AI world is NOT mutated by the render (a pure read — two calls byte-equal).
- **The render is the EXISTING float path** (the lit-instanced + transparent + debug-line pipelines, REUSED VERBATIM —
  NO new shader/RHI): from a fixed 3/4 camera + directional light over the ground. The agents run CPU-side (the AI
  lockstep is pure-CPU render-prep; the render is a normal draw, no compute, no TDR).
- **The bit-exact part = the AI world** (integer, AI1–AI5); the marker transforms + the final raster are float
  (render-only). So the golden is the FLOAT bar: a per-backend Metal-baked golden, Metal==Metal-golden DIFF 0.0000
  (deterministic two-run) + provenance (the markers/corridors derive from the bit-exact AI5 world) + a DOCUMENTED
  in-band cross-vendor mean (~40–60, the float baseline — the VD6/NAV6/IK6 convention).
- **The public sample (the issue's ask):** ship `samples/ai_demo/` (or document the `--ai6-render` showcase as the
  public AI sample) — an NPC that senses, decides, queries, paths, and acts, rendered, deterministic + replayable.
- **MAC-COMPILE:** `ai.h` is header-only; the `--ai6-render` handler transitively uses `verdict`/`nav` (header) +
  `animation.cpp`-style sources that are ALREADY in `metal_headless/CMakeLists.txt`; if a NEW `.cpp` is pulled, ADD it
  to that list (the consol-#65 lesson) and confirm a clean Mac build. Grep your OWN `visual_test.mm` for `--ai6-render`.
