> Autonomous-session spec. Verifiable headlessly on Windows+Vulkan (RTX 4070) + Mac+Metal. The THIRD slice of
> FLAGSHIP #32 (DETERMINISTIC IK CONTROL-RIG, slice-tag **IK**). IK1 retired the transcendental crux (the angle LUT);

> Both in Q16.16, bit-identical CPU/Vulkan/Metal. FABRIK is iterative → its end-effector residual is the documented
> within-band caveat (the Gauss-Seidel residual shape). **APPENDS to `engine/anim/ik.h`** (`hf::anim::ik`, IK1/IK2
> byte-frozen — append-only). Branch: `slice-ik3`. See [[hazard-forge-ik-roadmap]], [[hazard-forge-docs-style]],

## Design call: FABRIK = pure position reaching (no transcendentals); look-at = acos(dot)+cross (non-negative angle, the IK2 lesson)
- **FABRIK (Q16.16, pure integer, NO transcendental):** for a chain `pos[0..K-1]` with fixed segment lengths
  `len[i] = |pos[i+1] − pos[i]|` (captured at rest), a fixed `root`, and a `target` for the end-effector
  `pos[K-1]`, each iteration is two passes:
  - **Backward** (reach toward target): `pos[K-1] = target`; for `i = K-2 .. 0`: `pos[i] = pos[i+1] + len[i] *
    FxNormalize(pos[i] − pos[i+1])` (place each joint exactly `len[i]` from its successor along the current
    direction).
  - **Forward** (re-anchor to root): `pos[0] = root`; for `i = 1 .. K-1`: `pos[i] = pos[i-1] + len[i-1] *
    FxNormalize(pos[i] − pos[i-1])`.
  - Repeat `iters` times. Segment lengths are preserved by construction (each joint placed exactly `len` away — the
    only drift is the `FxNormalize` LSB error). If the target is unreachable (`> Σlen`), the chain straightens toward
    it deterministically. Pure `FxNormalize` (int64 sqrt) + integer position updates — NO `acos`/`sin`/`cos`.
  - **Determinism by construction:** fixed pass order, fixed `iters`, in-place integer mutation → two-run + GPU==CPU
    bit-identity (the cloth/Gauss-Seidel discipline). Single-thread `[numthreads(1,1,1)]` (the JT1 iterative mirror);
    chunk 1 iteration-set per dispatch ONLY if a heavy chain risks the TDR watchdog (a short chain × modest `iters`
    is light — a single dispatch is fine; document the choice).
- **Look-at (shortest-arc):** `to = FxNormalize(target − origin)`; `c = FxDotV(fwd, to)` (cosine, in `[-kOne, kOne]`);
  `angle = FxAcosLut(c)` (ALWAYS NON-NEGATIVE — `acos` returns `[0, π]`, so the IK2 negative-half-angle GPU==CPU
  divergence CANNOT arise here, the cleanest LUT path); `axis = FxNormalize(FxCross(fwd, to))`; `LookAtRotation =
  QuatFromAxisAngle(axis, angle)`. **Antiparallel fallback:** when `fwd ≈ -to` (`c ≈ -kOne`, the cross degenerates),
  pick a deterministic perpendicular axis (e.g. cross with world-up, else world-right) and rotate by `π`. When
  `fwd ≈ to` (`c ≈ kOne`), return identity.
- **The within-band caveat (honest):** FABRIK's end-effector reaches the target within a bounded, deterministic
  residual (the iterative Gauss-Seidel residual + `FxNormalize` LSB — NOT zero); segment lengths drift only by the
  `FxNormalize` LSB. The headline is DETERMINISM + cross-platform bit-identity, NOT analytic convergence. Document
  both bands.
