# Slice IK2 — Deterministic IK Control-Rig: DETERMINISTIC TWO-BONE IK (the law-of-cosines limb solve) — Design

> Autonomous-session spec. Verifiable headlessly on Windows+Vulkan (RTX 4070) + Mac+Metal. The SECOND slice of
> FLAGSHIP #32 (DETERMINISTIC IK CONTROL-RIG, slice-tag **IK**). IK1 retired the transcendental crux (the host-snapped
> Q16.16 `acos`/`sin`/`cos` LUT). IK2 builds the canonical limb solver on it: the two-bone (upper-arm/forearm,
> thigh/shin) analytic IK — given a root, a target, a pole/hint, and the two bone lengths, solve the elbow/knee bend
> via the law of cosines (the ONE `acos`, from IK1's LUT) and emit the two bone LOCAL rotations as `FxQuat`s, all in
> Q16.16 fixed-point, bit-identical CPU/Vulkan/Metal. This is the limb primitive the rig (IK4) + the lockstep headline
> (IK5) + the render capstone (IK6) drive. **APPENDS to `engine/anim/ik.h`** (`hf::anim::ik`, IK1 byte-frozen —
> append-only). Branch: `slice-ik2`. See [[hazard-forge-ik-roadmap]], [[hazard-forge-docs-style]],
> [[hazard-forge-build-env-vcvars]], [[hazard-forge-metal-showcase-gate]], [[hazard-forge-gpu-tdr-chunking]].

**Goal:** Append to `engine/anim/ik.h`: `struct IkChain2 { FxVec3 root; fx lenUpper; fx lenLower; }` (or pass as args)
and `TwoBoneSolve(root, target, pole, lenUpper, lenLower) -> { FxQuat qUpper; FxQuat qLower; }` (the two bone LOCAL
rotations), computing: `d = FxLength(target - root)` clamped to the reachable band `[|lenU - lenL|, lenU + lenL]`; the
root angle `acos((lenU² + d² − lenL²) / (2·lenU·d))` and the elbow interior angle `acos((lenU² + lenL² − d²) /
(2·lenU·lenL))` (both via `FxAcosLut`); build the bend-plane from the root→target axis + the pole, and synthesize the
two rotation quaternions from axis-angle (`FxSinLut`/`FxCosLut` of the half-angles). Reachable + **over-extended**
(`d ≥ lenU + lenL` → straight limb at the target, deterministic clamp) both deterministic. Add `shaders/ik_twobone.comp.hlsl`
(int64 → Vulkan-only; Metal runs the CPU `TwoBoneSolve` byte-identical). Add `--ik2-twobone-shot` (Vulkan) /
`--ik2-twobone` (Metal). Bake the **strict-zero** golden `ik2_twobone`.

## Design call: law of cosines + axis-angle quats; clamp the reach; the LUT-acos residual is the within-band caveat
- **The solve (Q16.16):** `e = target − root`; `d = FxLength(e)` (int64 sqrt). Clamp `d` to `[|lenU−lenL|, lenU+lenL]`
  (un-reachable-near and over-extended both clamp to the boundary → deterministic). The cosine arguments use `fxdiv`
  (int64): `cosRoot = fxdiv(lenU² + d² − lenL², 2·lenU·d)`, `cosElbow = fxdiv(lenU² + lenL² − d², 2·lenU·lenL)`
  (the squares/products formed in int64 then narrowed — the `FxLength` int64 discipline). `aRoot = FxAcosLut(cosRoot)`,
  `aElbow = FxAcosLut(cosElbow)`.
- **The plane + the quats:** `axis = FxNormalize(e)` (root→target). `n = FxNormalize(cross(axis, pole))` — the bend-plane
  normal (the rotation axis for both bones); if `cross` degenerates (pole ∥ axis) use a deterministic fallback axis.
  The upper bone rotates from `axis` by `aRoot` about `n`; the lower bone rotates from the upper by `π − aElbow` about
  `n`. Build each quaternion from axis-angle: `q = (n · FxSinLut(θ/2), FxCosLut(θ/2))` (the `FxQuat` w-last layout,
  fpx.h:109). The OUTPUT is the two bone LOCAL `FxQuat`s (feeding `anim::JointPose.r`, animation.h:38).
- **Over-extended → straight limb:** when `d ≥ lenU + lenL` (clamped), `aElbow = 0` (straight) and the limb points
  directly at the target — the deterministic, falsifiable clamp beat.
- **The within-band caveat (honest):** the end-effector will NOT hit the target EXACTLY — the `acos` LUT carries IK1's
  documented LSB band, so forward-kinematics from the solved quats reaches the target within a bounded, deterministic
  residual (NOT zero). The headline is DETERMINISM + cross-platform bit-identity, NOT analytic reach. Document the band.
- **A `cross(a,b)` helper:** if `fpx.h` has no `FxCross`, add a local `inline FxVec3 FxCross(...)` in `ik.h` (int64
  products narrowed to Q16.16 — the `FxLength` discipline). Grep fpx.h first; reuse if present.
